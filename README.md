# PH-HSP

Simple data pipeline analyzing hospital capacity vs population in Philippine municipalities.

## What it does
- Cleans hospital and population datasets  
- Stores structured data in PostgreSQL  
- Computes hospital beds per 1,000 people  
- Compares healthcare capacity across municipalities  

## Tech stack
- SQL (PostgreSQL)
- Python (pandas, numpy)
- Git / GitHub

## Output
- Clean population table  
- Clean hospital table  
- Beds per 1,000 population metric  

## Result
Basic healthcare capacity analysis across municipalities in Catanduanes.
